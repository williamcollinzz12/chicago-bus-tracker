# CTA Bus Tracker

A retro-styled real-time bus tracker for Chicago's CTA system.

## Features
- Live bus arrival predictions
- Real-time bus location tracking on map
- Push notifications for approaching buses
- Vintage transit aesthetic
- Works as a progressive web app (PWA)

## Usage
Visit: https://williamcollinzz12.github.io/chicago-bus-tracker/

Add to your iPhone home screen for app-like experience.

## Setup
Requires a CTA Bus Tracker API key (free from transitchicago.com)
